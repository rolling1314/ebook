## 🔧 重要技术决策

### 1. EPUB 文件认证
**问题**：epub.js 无法在请求中发送 Bearer token

**解决方案**：支持 query parameter token
```
GET /api/books/{book_id}/file.epub?token={jwt_token}
```

### 2. 笔记显示方式
**选择**：Kindle 风格内嵌显示

**原因**：
- 更直观的阅读体验
- 笔记紧跟高亮文本
- 符合用户习惯

### 3. 数据库选择
**选择**：PostgreSQL

**原因**：
- 成熟稳定
- 支持复杂查询
- 完整的事务支持
- 良好的 Python 生态
